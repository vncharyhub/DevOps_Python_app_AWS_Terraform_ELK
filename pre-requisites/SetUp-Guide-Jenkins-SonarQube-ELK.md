
## Required configuration for Jenkins, SonarQube, ELK stack

### Minimum Viable Setup

| Service      | Instance Type  |
|--------------|----------------|
| Jenkins VM   | `t2.micro`     |
| SonarQube VM | `t2.medium`    |
| ELK VM       | `t2.medium`    |

### Configuration Required

| --------- | ----------- | --- | ------- | ------------------------------- |
